Dichotischer Hörtest

Dies ist der letzte Hörtest.

Für diesen Test benötigen Sie Kopfhörer. Bitte setzen Sie diese auf.

Sie werden gleichzeitig auf beiden Ohren verschiedene, aber bekannte Wörter hören.

Merken Sie sich beide und tragen Sie sie ohne Artikel in die dafür vorgesehenen Felder ein. Es ist dabei unerheblich in welches Feld Sie die Wörter eintragen. Sollten Sie ein Wort nicht verstanden haben, so lassen Sie das Feld einfach frei.

Lassen Sie die Kinder Ihnen beide Wörter zum Eintragen vorsprechen.

Drücken Sie 'Start', um den Test zu beginnen.

Geben Sie die beiden gehörten Wörter an:

Vielen Dank für Ihre freundliche Unterstützung!

Auf Ihrem Desktop finden Sie folgende PDF-Dokumente:

"Anamnese und Testergebnisse.pdf",

"Mottier-Test.pdf"

"Diskrimination.pdf"

und "Dichotisch.pdf".

Bitte bringen Sie diese ausgedruckt zu Ihrem Besuch bei uns mit!

Auf diese Weise tragen Sie zu einem reibungslosen Ablauf Ihres Besuchs bei.
Herzlichen Dank!
